# Financial Editorial - Subscription Tracker

## Getting Started

The project files have been completely scaffolded with Next.js App Router, Tailwind CSS, Framer Motion, and the requested UI pages (Dashboard, Analytics, List, Add, Settings, Login, Signup).

1. **Install Dependencies:**
   Run `npm install` in your terminal to install React, Next.js, Tailwind, Framer Motion, and Supabase.
   
2. **Start the Development Server:**
   Run `npm run dev`. Navigate to `http://localhost:3000` to see the application.

3. **Deploy to Vercel:**
   - Commit these files to a GitHub repository.
   - Go to Vercel and import the repository.
   - Vercel will automatically detect the Next.js configuration and deploy it for you.

## Database
- Use the `supabase_schema.sql` file in your Supabase SQL Editor to instantly create the `subscriptions` table and security policies.
